Para criar o compilador executa-se o comando make.

Daqui resulta um executável com nome compiler.

O executável compiler compilará o ficheiro que for passado como argumento.

Por exemplo:

./compiler programa

-------------------------------------------------------------------
Código intermédio:

O código intermédio gerado será imprimido no terminal.

-------------------------------------------------------------------
Código assembly:

O código assembly gerado será guardado no ficheiro code.asm

Esse código foi posteriormente testado utilizando o simulador de instruções mips Mars.
